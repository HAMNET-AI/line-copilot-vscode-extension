## Usage

- 安装需要的包

  ```bash
  npm install
  npm install vsce -g
  ```

- 使用 `vsce` 进行打包

  ```bash
  vsce package
  ```

- 直接从 VSCode 中安装 `vsix` 文件

- 在插件设置中，填写 BOOK_ID（API_KEY 暂时不用设置）

- 输入文字自动触发原文补全
